# Shivsahyadri Pratishthan Website 🚩

Welcome to the official repository of the **Shivsahyadri Pratishthan** web platform. This project is built to showcase the organization's rich history, multi-year social projects, and to provide an easy way for well-wishers to support the cause through an integrated donation portal.

---

## 🚀 Features

- **Interactive UI/UX:** Completely redesigned, clean, modern, and responsive user interface.
- **Chronological Project Archives:** Dedicated sections showcasing milestone initiatives from past years:
  - Project 2019-20
  - Project 2021-22
  - Project 2022-23
  - Project 2023-24
  - Project Vijay
- **Donation Portal:** A secure and user-friendly donation page (`donate.html`) allowing supporters to contribute to ongoing social welfare projects.
- **Dynamic Frontend:** Powered by vanilla JavaScript (`shivsahyadri.js`) for smooth transitions and interactivity.
